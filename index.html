User Journey Map · PMB Affiliate

Konfigurasi Afiliasi PMB
di SIAKAD 4.0

Gambaran rancangan flow langkah demi langkah konfigurasi hingga fitur afiliasi siap digunakan oleh kampus.

Aktor
Admin Suteki
Sistem SIAKAD
Admin Kampus (PMB)
Fase 1 Akses & Persiapan
1
Admin Suteki
Buka Halaman Konfigurasi

Admin Suteki masuk ke panel internal SIAKAD dan membuka halaman khusus pengaturan fitur Afiliasi PMB. Halaman ini hanya bisa diakses oleh tim Suteki (bukan oleh pihak kampus).

🔒 Akses terbatas oleh Admin Suteki 📍 Panel Internal SIAKAD
2
Sistem Otomatis
Sistem Membaca Daftar Prodi

Begitu halaman terbuka, sistem langsung mengambil data dari database SIAKAD, tanpa perlu klik tombol apa pun. Semua program studi aktif yang tercatat di SIAKAD kampus tersebut langsung ditampilkan.

⚡ Otomatis saat halaman dibuka 📋 Semua prodi yang terdata di sistem ditampilkan
Fase 2 Cek & Isi Konfigurasi
3
Sistem Otomatis
Cek Kelengkapan Konfigurasi

Sistem memeriksa apakah semua prodi sudah terisi. Jika ada yang belum, muncul banner peringatan yang menyebutkan jumlah prodi yang masih kosong.

Contoh peringatan: "3 program studi belum dikonfigurasi. Harap lengkapi sebelum menyimpan."
4
Admin Suteki
Isi Konfigurasi per Prodi

Untuk setiap prodi, admin mengisi tiga informasi wajib:

💰 Fee Bruto: komisi total untuk afiliator (Rp) ⏱ Trigger Pencairan: kondisi penentuan dihitungnya komisi ✂️ Potongan Suteki (read only) : biaya platform (% sesuai dengan kesepakatan)
Opsi trigger: Daftar · Lulus Seleksi · Daftar Ulang
Fase 3 Validasi & Simpan
5
Sistem Otomatis
Validasi Input

Saat admin mencoba menyimpan, sistem memvalidasi setiap field. Jika ada yang tidak valid, pesan error muncul langsung di bawah field yang bermasalah. Admin harus memperbaiki dulu sebelum bisa lanjut.

Contoh error: "Trigger belum dipilih"
Jika semua valid, sistem mengizinkan proses penyimpanan dilanjutkan.
6
Admin Suteki
Simpan Konfigurasi

Setelah semua terisi dan valid, admin klik Simpan. Konfigurasi untuk setiap prodi: fee bruto, trigger pencairan, dan potongan Suteki, tersimpan ke database SIAKAD.

💾 Tersimpan per prodi di database SIAKAD
Fase 4 Sinkronisasi Otomatis
7
Sistem Otomatis
Sinkronisasi ke Portal PMB

Konfigurasi yang sudah tersimpan langsung terbaca oleh portal PMB kampus. Tidak perlu upload atau import manual. Prodi baru otomatis muncul; prodi yang dinonaktifkan di SIAKAD otomatis tersembunyi.

SIAKAD
Konfigurasi tersimpan
→
Portal PMB
Afiliasi → Pengaturan Program
real-time, otomatis
Fase 5 Siap Digunakan Kampus
8
Admin Kampus (PMB)
Fitur Afiliasi Siap Dikelola

Admin PMB kampus sudah bisa membuka menu Afiliasi → Pengaturan Program dan mengelola program afiliasi sesuai konfigurasi yang sudah disiapkan Suteki. Sistem PMB juga sudah bisa melakukan pengecekan komisi afiliasi secara otomatis berdasarkan prodi dan trigger yang berlaku.

✅ Prodi aktif tersedia di daftar afiliasi 🔍 Lookup komisi berjalan otomatis
🎯

Hasil Akhir

Setelah proses ini selesai, kampus dapat langsung menggunakan fitur afiliasi PMB. Setiap afiliator yang mendaftarkan calon mahasiswa akan tercatat pendapatan komisi sesuai aturan yang telah dikonfigurasi dan dihitung otomatis oleh sistem tanpa perlu input manual dari kampus.
